# Index Builder - 使用说明

## 📦 快速开始

### Windows 用户

1. **双击 `启动.bat`**
2. 选择 `[1] 管理面板`（推荐）
3. 浏览器会自动打开管理界面

### Linux/Mac 用户

1. **在终端运行**:
   ```bash
   chmod +x 启动.sh
   ./启动.sh
   ```
2. 选择 `[1] 管理面板`（推荐）

---

## ⚙️ 首次使用配置

### 1. 配置 LLM API (可选)

如果需要使用 AI 增强功能：

1. 编辑 `config/.env` 文件
2. 配置你的 API 密钥：
   ```
   LLM_API_BASE=https://api.openai.com/v1
   LLM_API_KEY=你的-api-密钥-here
   LLM_MODEL=gpt-4o-mini
   ```

### 2. 准备数据

确保以下目录结构存在：

```
data/
├── raw/              # 原始数据
│   ├── raw.csv       # 原始查询数据
│   └── intent_seed.jsonl  # 意图种子数据
├── dict/             # 词典文件
│   ├── stopwords.txt
│   ├── user_dict.txt
│   └── normalize_regex_map.txt
└── processed/        # 处理后的数据（自动生成）
```

## 🎯 功能模块

### 🏠 管理面板 (Dashboard)

统一管理界面，包含：
- 环境状态检查
- 快速启动各功能
- Pipeline 流程管理
- 数据文件浏览
- 配置文件编辑

**启动方式**: `启动.bat` → 选择 [1]

---

### 📋 审核界面 (Review App)

用于人工审核自动分类结果：
- 查看系统推荐的意图分配
- 人工确认或修改
- 批量处理和进度跟踪

**启动方式**:
- 管理面板 → 快速启动 → 启动审核界面
- 或 `启动.bat` → 选择 [2]

---

### 🛠️ 工作台 (Workbench)

意图目录管理与优化：
- 混淆分析：识别容易混淆的意图
- 混淆优化：编辑和调整意图定义
- 新增意图：辅助创建新意图
- AI 建议：使用 LLM 提供优化建议

**启动方式**:
- 管理面板 → 快速启动 → 启动工作台
- 或 `启动.bat` → 选择 [3]

---

## 🔄 完整工作流程

### Pipeline 步骤

```
1. 准备目录
   ↓
2. 扩展目录 (可选 LLM 增强)
   ↓
3. 构建字段
   ↓
4. 分配查询 (BM25 自动分配)
   ↓
5. 人工审核 (使用审核界面)
   ↓
6. 收集反馈
```

### 详细操作

1. **在管理面板中，进入「⚙️ Pipeline」页面**
2. **按顺序执行每个步骤**
3. **第5步时，使用「快速启动」打开审核界面**
4. **完成审核后，继续执行后续步骤**

## 🔧 故障排除

### 问题：Python 未找到

**解决方案**:
1. 下载安装 Python 3.9 或更高版本
2. 安装时勾选 "Add Python to PATH"
3. 重启命令行窗口

### 问题：依赖安装失败

**解决方案**:
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 问题：端口被占用

**解决方案**:
- 启动器会自动寻找可用端口
- 或手动修改 streamlit 配置

### 问题：浏览器没有自动打开

**解决方案**:
- 手动复制控制台显示的 URL
- 在浏览器中打开

如果启动器无法使用，可以手动运行：

```bash

streamlit run launcher/dashboard.py

# 审核界面

```
